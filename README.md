# Mi Primera Web

Proyecto open source pensado para ayudar a personas con poca experiencia
técnica a crear una página web sencilla, modificarla, guardarla en GitHub y
publicarla paso a paso.

## Objetivo

La idea no es vender un curso ficticio, sino ofrecer una base práctica para
aprender:

- la estructura básica de una web en HTML
- cómo cambiar textos, imágenes y estilos
- cómo usar Git y GitHub para guardar versiones
- cómo compartir y publicar una página sencilla

## Estructura del proyecto

- `index.html`: contenido principal de la página
- `styles.css`: estilos visuales
- `desarrollador-software-ia-entorno-futurista.webp`: recurso gráfico disponible

## Cómo abrirlo en local

Al ser una web estática, puedes abrir `index.html` directamente en el
navegador.

Si prefieres servirlo desde un servidor local, puedes usar por ejemplo:

```bash
python3 -m http.server 8000
```

Y después abrir `http://localhost:8000`.

## Enfoque

Este repositorio está orientado a:

- alumnado
- docentes
- personas que empiezan con programación
- autónomos o pequeños equipos
- cualquiera que quiera entender cómo se construye una web simple

## Uso recomendado

1. Clona o descarga el proyecto.
2. Modifica la plantilla.
3. Guarda cambios con Git.
4. Sube el proyecto a GitHub.
5. Publica tu página y sigue iterando.
